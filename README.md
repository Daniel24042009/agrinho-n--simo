# agrinho-n--simo
Plataforma digital interativa desenvolvida para a Subcategoria 3 (Programação Front-End) do Concurso Agrinho 2026.
# Raízes do Amanhã - Inteligência Sustentável

## Objetivo do Tema Agrinho 2026
Este projeto foi desenvolvido para a Subcategoria 3 (Programação Front-End) do Concurso Agrinho 2026, com o tema: "Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente". O objetivo é oferecer uma plataforma digital que monitora e incentiva práticas agroecológicas de forma interativa.

## Tecnologias Utilizadas
- **HTML5:** Estruturação semântica em conformidade rigorosa com o edital (sem CSS ou JS inline).
- **CSS3:** Estilização responsiva totalmente autoral, sem recurso a frameworks ou bibliotecas externas.
- **JavaScript (Vanilla):** Lógica de interatividade estruturada e manipulação inteligente do DOM.

## Instruções de Uso
1. Transfira (faça o download) deste repositório para o seu computador.
2. Abra o ficheiro `index.html` em qualquer navegador web (ex: Google Chrome, Firefox ou Edge).
3. Utilize o menu lateral para navegar pelas secções: "Painel Inicial", "Agroecologia", "Lições do Campo", "Materiais de Apoio", "Jogo da Memória" e "Meu Progresso".
